SENSORS


Sensors play an important role in creating solutions using IoT. Sensors are devices that collect external information and send that data to the machine for computation.

ACTUATORS


An actuator is a device that converts energy into motion. It does this by taking an electrical signal and combining it with an energy source.

Development board


Development Boards are printed circuit boards with a microcontroller/microprocessor mounted on them with few other hardware components. We can consider development board as a foundation for IoT.

Sensors

DHT

Temperature humidity sensor

The DHT11 is a basic, ultra low-cost digital temperature and humidity sensor. It uses a capacitive humidity sensor and a thermistor to measure the surrounding air, and spits out a digital signal on the data pin (no analog input pins needed).

Work on it!
Ultrasonic sensor

Used to measure distance of the obstacle

An ultrasonic sensor is an instrument that measures the distance to an object using ultrasonic sound waves. An ultrasonic sensor uses a transducer to send and receive ultrasonic pulses that relay back information about an object's proximity.

Work on it!
IR sensor

Used to know if object is present

An infrared sensor (IR sensor) is a radiation-sensitive optoelectronic component with a spectral sensitivity in the infrared wavelength range 780 nm … 50 µm.

LDR

Used to know the light intensity

Light dependent resistors (LDR), are light sensitive devices most often used to indicate the presence or absence of light, or to measure the light intensity.

HC05

Bluetooth sensor

Designed to replace cable connections HC-05 uses serial communication to communicate with the electronics. Usually, it is used to connect small devices like mobile phones using a short-range wireless connection to exchange files.
